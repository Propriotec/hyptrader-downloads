# Hyptrader Downloads

Public mirror of installable release artifacts for **[Hyptrader](https://hyptrader.com)**, the desktop trading terminal by [Propriotec](https://github.com/Propriotec).

This repository hosts only the built installers — source code lives in the private `forex-platform` monorepo.

## Links

- **Website**: [hyptrader.com](https://hyptrader.com)
- **Trading platform**: [app.hyptrader.com](https://app.hyptrader.com)
- **Latest release**: [Releases page](https://github.com/Propriotec/hyptrader-downloads/releases/latest)

## Download

Grab the latest build for your platform from the [Releases page](https://github.com/Propriotec/hyptrader-downloads/releases/latest).

| Platform | Architecture | File |
| --- | --- | --- |
| macOS | Apple Silicon (M-series) | `hyptrader-macos-arm64.dmg` |
| macOS | Intel | `hyptrader-macos-x64.dmg` |
| Windows | x64 | `hyptrader-setup.exe` |
| Linux | x64 (Debian/Ubuntu) | `hyptrader-linux-x64.deb` |
| Linux | x64 (Fedora/RHEL) | `hyptrader-linux-x64.rpm` |
| Linux | x64 (portable) | `hyptrader-linux-x64.AppImage` |

## Install

- **macOS**: open the `.dmg`, drag **Hyptrader** to Applications.
- **Windows**: run the `.exe` installer and follow the prompts.
- **Linux (deb)**: `sudo apt install ./hyptrader-linux-x64.deb`
- **Linux (rpm)**: `sudo dnf install ./hyptrader-linux-x64.rpm`
- **Linux (AppImage)**: `chmod +x hyptrader-linux-x64.AppImage && ./hyptrader-linux-x64.AppImage`
